\doxysubsection{Detailed Description}
This tool generates a graph starting from a seed. 

This tool uses an interger n to generate a graph with 2$^\wedge$n replicas of the seed keeping all the edges already present in the seed, in addition edges are added between the different seeds of the final graph following the probability passed. ~\newline
The first parameter is the path of seed graph. ~\newline
The second parameter is the path of graph generated. ~\newline
The thrid parameter is an integer n that indicate the 2$^\wedge$n copy to be created. ~\newline
The fourth parameter is the probability of create an edge between a node of a copy and another and viceversa. ~\newline
 Es\+: seed 10 edge and second parameter 1 graph generated 20 edge. ~\newline
 ~\newline
 Es\+: seed 10 edge and second parameter 2 graph generated 40 edge. ~\newline
 ~\newline
 Es\+: seed 10 edge and second parameter 3 graph generated 80 edge. ~\newline
 ~\newline
 Es\+: seed 10 edge and second parameter 4 graph generated 160 edge. ~\newline
